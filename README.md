# Substrate .NET Discord Bot

## How to use
- Get a Discord Token by following this steps https://discordnet.dev/guides/getting_started/first-bot.html
- Replace in `config.yml` `<DISCORD_TOKEN_HERE>` with your token
- Run the bot
- execute `/node-block`
